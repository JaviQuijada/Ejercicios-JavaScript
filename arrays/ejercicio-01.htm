Ejercicio 1 Saludos

Estructuras de datos: Arrays

Crear un programa que pida ingresar nombres separados por espacios y devuelva un mensaje que contenga un saludo a cada persona. Ejemplo:
```javascript
// Ingrese nombres: Ada Greta Grace
// ¡Hola Ada!
// ¡Hola Greta!
// ¡Hola Grace!
```
