Ejercicio 21 Mayor número

Arrays 2D

Crear un programa que dado un array 2d de números, indique cuál es el mayor de ellos. El array debe estar guardado en una variable (no hace falta ingresarlo). Array de ejemplo:
```javascript
[
[2, 7, 12, 1],
[8, 23],
[9, 45, 7],
[22, 3, 24, 4]
]
```
